# graphify output (optional)

This directory holds **generated** graphify artifacts when you run `graphify` / `make graphify-update` from the repo root.

- Prefer reading **`GRAPH_REPORT.md`** here (if present) before large architectural changes.
- Safe to add `graph.json` and other outputs to `.gitignore` if they grow large; this stub exists so the folder is discoverable in fresh clones.
